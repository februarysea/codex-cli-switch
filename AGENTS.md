# AGENTS.md

Project guidance for coding agents working in this repository.

- Keep changes focused on the requested task and preserve the existing Bash-first style.
- Prefer `rg` / `rg --files` for searching files and text.
- Use `bash -n` for shell syntax checks after editing shell scripts.
- Do not overwrite or revert user changes unless explicitly asked.
- After every intentional file modification, stage the changed file with `git add <path>`.
